# SonicWall Firewall & VPN Lab

## Overview
This repository documents a network security and remote access support lab focused on SonicWall firewall concepts, SSL VPN/NetExtender access, firewall rules, NAT, access groups, logs and troubleshooting.

## Purpose
The goal is to show practical firewall and VPN support knowledge that supports MSP, systems support, cloud support and network support roles.

## Focus Areas
- Firewall rule logic and traffic control
- WAN, LAN and VPN access concepts
- SSL VPN / NetExtender support workflow
- User group membership for SSLVPN/SSL-VPN access
- NAT concepts and protected internal resources
- Log review and connection troubleshooting
- Documentation of remote access support steps

## Example Troubleshooting Scenario
**Issue:** User cannot connect through NetExtender / SSL VPN.

**Checks:**
1. Confirm the exact error message and whether the issue affects one user or multiple users.
2. Confirm the user has working internet access.
3. Confirm username/password/MFA status if identity checks are required.
4. Confirm the user is in the correct SSLVPN/SSL-VPN group.
5. Check VPN/firewall logs for authentication failure or policy denial.
6. Confirm whether the user can reach the required internal resource after connection.
7. Document the root cause and resolution.
