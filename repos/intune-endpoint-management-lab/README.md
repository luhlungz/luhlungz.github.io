# Intune Endpoint Management Lab: Compliance, App Deployment & Device Management

## Overview
This lab demonstrates Windows endpoint management using Microsoft Intune.

## Goal
Build practical evidence for Endpoint Support, Intune Support, Microsoft 365 Administrator and MD-102-aligned roles.

## Tools Used
- Microsoft Intune
- Microsoft Entra ID
- Windows client device
- Company Portal
- Defender for Endpoint
- Application deployment

## What I Tested
1. Windows device enrollment into Intune.
2. Device sync and check-in status.
3. Windows compliance policy assignment.
4. Company Portal behaviour.
5. 7-Zip app deployment.
6. Install status monitoring.
7. Defender onboarding status.
8. Policy validation and troubleshooting.

## Troubleshooting Checklist
- Confirm the device is enrolled.
- Confirm MDM user scope.
- Confirm assignment target group.
- Sync device from Intune and from the client.
- Check install status and error messages.
- Review device compliance state.
