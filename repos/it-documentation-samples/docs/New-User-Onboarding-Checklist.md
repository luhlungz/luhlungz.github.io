# New User Onboarding Checklist

## Purpose
Provide a repeatable process for preparing a new user account, mailbox, device and access requirements.

## Required Information
- Full name
- Job title
- Department
- Manager
- Start date
- Required groups
- Required mailbox access
- Required applications
- Required VPN access

## Steps
1. Confirm approval from the requester or manager.
2. Create or verify the user account.
3. Assign the correct Microsoft 365 license.
4. Add the user to required groups.
5. Configure MFA registration requirement.
6. Configure mailbox permissions if required.
7. Prepare device and install required applications.
8. Confirm VPN access if required.
9. Document completed work in the ticket.
10. Send handover confirmation.

## Validation
- User can sign in.
- License is active.
- Mailbox is accessible.
- MFA registration is complete.
- Required apps are available.
- Required groups are correct.
