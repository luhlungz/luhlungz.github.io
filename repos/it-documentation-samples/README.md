# IT Documentation Samples

## Overview
This repository contains sanitised documentation samples for MSP-style support and administration work.

## Documentation Samples To Add
- New user onboarding checklist
- User offboarding checklist
- Shared mailbox access SOP
- MFA reset / registration SOP
- Intune app deployment SOP
- Device handover checklist
- VPN access support checklist

## Recommended Structure For Each SOP
1. Purpose
2. When to use this process
3. Required access / approvals
4. Step-by-step process
5. Validation checks
6. Escalation notes
7. Documentation notes

## Example: Shared Mailbox Access SOP
### Purpose
Provide a repeatable process for granting and validating shared mailbox access.

### Steps
1. Confirm approval.
2. Confirm user and mailbox.
3. Apply Full Access, Send As or Send on Behalf as required.
4. Ask user to test Outlook on the web.
5. Document permission and outcome.
