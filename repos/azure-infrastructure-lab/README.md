# Azure Infrastructure Lab: VNet, VMs, NSG, ASG & Monitoring

## Overview
This lab demonstrates a basic Azure infrastructure environment with networking, Windows Server VMs, security rules and basic governance.

## Goal
Build practical evidence for Cloud Support Engineer, Junior Azure Administrator and Systems Support roles.

## Tools Used
- Microsoft Azure
- Azure Resource Groups
- Azure Virtual Network
- Subnets
- Azure Virtual Machines
- Network Security Groups
- Application Security Groups
- Windows Server
- IIS
- Azure Activity Log Alerts
- Azure Budgets
- Azure Resource Locks

## What I Built
1. Created Azure resource group.
2. Created VNet and subnet.
3. Deployed Windows Server VM AZSRV01.
4. Installed IIS and tested HTTP access.
5. Deployed Windows Server VM AZSRV02.
6. Configured NSG rules.
7. Used ASGs for role-based traffic control.
8. Restricted RDP access to AZSRV02 from AZSRV01 only.
9. Tested connectivity and network rules.
10. Configured budget, locks and activity log alerts.
