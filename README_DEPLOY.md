# Deployment Guide

This package uses the original portfolio design and includes actual lab screenshots.

1. Extract the ZIP.
2. Open the `lungani-portfolio` folder.
3. Upload the CONTENTS of that folder to the root of `luhlungz/luhlungz.github.io`.
4. Confirm these files exist in the GitHub repo root:
   - `index.html`
   - `evidence.html`
   - `assets/Lungani_Zondi_CV.pdf`
   - `assets/evidence/azure/`
   - `assets/evidence/intune/`
   - `assets/evidence/defender-security/`
   - `assets/evidence/scripts-kql/`
5. Commit to `main`.

Suggested commit message:
Add lab evidence screenshots to portfolio
