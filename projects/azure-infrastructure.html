Azure Infrastructure Lab

Azure VNet, subnet segmentation, Windows Server VMs, NSG/ASG rules, Private DNS, RBAC, monitoring, Log Analytics, KQL, Policy and Resource Graph.

Project overview

Built an Azure lab to practise infrastructure deployment, network segmentation, private server access, role-based access control, monitoring and governance.

Portfolio value: This project shows practical evidence, not just theory. The screenshots below show the portal results, command results and validation checks.
